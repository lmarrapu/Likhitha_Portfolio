Atrial Fibrillation Detection

Developed an advanced Atrial Fibrillation (AF) detection system using ECG signals from the PhysioNet dataset, achieving an impressive 76.15% accuracy. The project involved implementing and optimizing multiple machine learning models, including Support Vector Machines (SVM), LightGBM, Convolutional Neural Networks (CNN), and a Hybrid model, to effectively handle class imbalance and improve classification performance. Key preprocessing steps included noise filtering, signal inversion correction, and feature extraction from both time-domain and frequency-domain characteristics. The models performance was thoroughly evaluated using metrics like F1-score and ROC-AUC, ensuring robust and reliable AF detection, with real-world applications in early diagnostics and remote monitoring.

Twitter Sentiment Analysis

This project focuses on performing sentiment analysis on 1.6 million tweets using Natural Language Processing (NLP) and machine learning techniques. A robust NLP pipeline was developed for effective text normalization, custom tokenization, and contextual lemmatization. Various machine learning models, including Logistic Regression, Naive Bayes, and Neural Networks, were built and optimized to classify sentiments as positive, negative, or neutral. The project achieved 78% accuracy through advanced techniques such as SMOTE for handling class imbalance and feature engineering with TF-IDF for enhanced text representation. The models were evaluated using metrics like accuracy, precision, recall, and F1-score to ensure reliable sentiment classification.

Hospitality Analytics Using Power BI

AtliQ Grands, a luxury hotel chain, faced declining revenue and market share due to rising competition and ineffective decision-making. To address this, an advanced Business Intelligence (BI) solution was developed using Power BI, providing real-time insights into Revenue Analytics, Booking Trends, and Guest Experience. The interactive dashboards uncover key business trends, such as revenue performance across cities, booking patterns, and guest satisfaction metrics, enabling data-driven strategies to optimize occupancy rates and enhance guest experiences. Insights revealed critical areas for improvement, including high cancellation rates, underperforming room categories, and reliance on third-party bookings. This solution empowers AtliQ Grands to refine its pricing strategy, improve direct bookings, and elevate customer satisfaction, driving sustainable business growth.
